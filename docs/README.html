📁 Structure

Project Layout

Expo Router file-based routing with a clean separation between screens, engine logic, and reusable components.

app/                        -- Expo Router screens
  _layout.tsx              -- Root layout (dark theme, Stack nav)
  index.tsx                -- Landing page (hero + feature cards)
  family.tsx               -- Family comparison (3 upload slots)
  celebrity.tsx            -- Celebrity match (2 upload slots)
  camera.tsx               -- FaceID-style camera scanner

src/
  engine/                  -- Face analysis (pure TypeScript)
    constants.ts           -- FEATURES, FEATURE_WEIGHTS
    landmarks.ts           -- normaliseLandmarks, featureSim, scores
    accuracy.ts            -- computeAccuracy, assessCameraQuality
    index.ts               -- barrel export

  components/
    Button.tsx             -- primary / secondary / ghost variants
    UploadCard.tsx         -- Photo upload + preview + quality badge
    GaugeChart.tsx         -- Animated SVG circular gauge
    FeatureBar.tsx         -- Animated horizontal bars + breakdown
    FaceOvalGuide.tsx      -- SVG oval overlay (FaceID-style)

  hooks/
    useImagePicker.ts      -- Generic picker with slot management

  theme/
    colors.ts              -- Dark glassmorphism palette
    spacing.ts             -- Spacing, radius, breakpoints

app.json                    -- Expo config
eas.json                    -- EAS Build profiles
tsconfig.json               -- TypeScript config
package.json                -- Dependencies + scripts
