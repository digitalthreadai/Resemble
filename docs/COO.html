Strategy & Features

Product vision, technical architecture, and roadmap for the Resemble face comparison platform. Built with Expo SDK 55 for iOS, Android, and Web from a single codebase.

Vision
Resemble uses AI to compare facial features between people — scoring resemblance with a hybrid engine combining neural embeddings (70%) and geometric landmark analysis (30%). 100% on-device processing — face data never leaves your phone.

Per-Feature Breakdown

Eyes, nose, mouth, jawline, and eyebrows scored independently with weighted contribution to overall similarity.

Family Triangulation

Three-way comparison between child and both parents, with per-parent inheritance percentages and combined analysis.

Deterministic Results

Same photos always produce the same scores. No cloud randomness or server-side variation. Reproducible and trustworthy.

Privacy First

All face processing runs on-device. No images uploaded, no data stored remotely, no accounts required.

Infrastructure

▼
⚙
Hybrid Scoring Engine
70% neural embedding similarity + 30% geometric landmark analysis. Weighted per-feature scoring with sigmoid kinship calibration.
🌐
Cross-Platform Detection
face-api.js on web, ML Kit + Vision Camera on native. Platform-agnostic landmark interface.
🔄
ML Kit Landmark Mapper
Translates ML Kit contour points to 68-point dlib-compatible landmarks for unified engine processing.
📈
Quality Assessment
Evaluates face size, centering, lighting, and landmark confidence. Blocks low-quality inputs from analysis.
📚
Onboarding
Animated walkthrough explaining modes, camera usage, and privacy guarantees on first launch.
Engine Architecture

Scoring Pipeline

Photo Input ↓ Platform Face Detector (face-api.js | ML Kit + Vision Camera) ↓ 68-Point Landmark Extraction → Normalise to unit square ↓ Per-Feature Similarity ├─ Eyes ............. 25% weight ├─ Nose ............. 25% weight ├─ Lips & Mouth ..... 20% weight ├─ Face Shape ....... 15% weight └─ Eyebrows ......... 15% weight ↓ Weighted Overall Score → Sigmoid Kinship Calibration → Final %

Feature Weights

FeatureWeightLandmarksNotes
Eyes25%12 pts (36-47)Eye shape & lid curvature
Nose25%9 pts (27-35)Bridge, tip & nostril width
Lips & Mouth20%20 pts (48-67)Lip shape, cupid's bow & width
Face Shape15%17 pts (0-16)Jawline width & chin shape
Eyebrows15%10 pts (17-26)Arch height & shape

Kinship Calibration

Raw cosine similarity (0.0 – 1.0) ↓ Sigmoid mapping: score = 1 / (1 + exp(-k * (raw - midpoint))) ├─ Family range: 0.30 – 0.60 raw → maps to intuitive 40% – 85% display └─ Prevents clustering around 50% — spreads scores into meaningful range ↓ Display percentage: calibrated for user comprehension
